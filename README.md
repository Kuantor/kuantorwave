# KuantorWave
High‑performance binary audio engine with a clean Python API for fast, reliable format conversion.

KuantorWave is a compiled audio‑processing core with a clean Python interface. It provides fast, reliable conversion between popular audio formats such as M4A ↔ MP3, MP3 ↔ WAV, and others. The core handles decoding, encoding, resampling, and metadata extraction, while Python acts as a lightweight orchestration layer.

KuantorWave is designed as the audio‑focused sibling of KuantorFlow, sharing the same philosophy:
a powerful binary engine with a simple, elegant API.
